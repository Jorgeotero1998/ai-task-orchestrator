# AI Task Orchestrator (Sistema AI-BIO)
**Desarrollado por Jorge Otero**

## Descripción del Proyecto
AI Task Orchestrator es una plataforma avanzada de orquestación diseñada para gestionar tareas complejas mediante inteligencia artificial. El sistema permite la coordinación eficiente entre modelos de procesamiento, persistencia de datos y una interfaz de usuario intuitiva.

## Cómo Funciona
El sistema opera mediante una estructura de tres capas:
1. **Capa de Orquestación (Backend):** Gestiona la lógica de negocio y la persistencia de datos mediante una API REST.
2. **Capa de Interfaz (Frontend):** Proporciona un panel de control interactivo en React.
3. **Capa de Infraestructura:** Utiliza contenedores Docker para un despliegue consistente.

## Tecnologías Utilizadas
### Backend
- **Lenguaje:** Python
- **Framework:** Flask
- **Testing:** Pytest

### Frontend
- **Lenguaje:** JavaScript (ES6+)
- **Librería:** React
- **Testing:** Jest

### Infraestructura y Automatización
- **Contenedores:** Docker y Docker Compose
- **Automatización:** Scripts de PowerShell

## Instalación y Ejecución
`powershell
docker-compose up --build
`
