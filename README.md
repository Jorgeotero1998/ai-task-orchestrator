## 🎯 ¿Qué es AI-BIO?
**AI-BIO** es un orquestador de tareas inteligente diseñado para la gestión estratégica y la optimización de flujos de trabajo. Utiliza inteligencia artificial de última generación para transformar objetivos ambiguos en hojas de ruta precisas y accionables.

## ✨ Funcionalidades Clave
- **Inferencia Ultra-Rápida:** Integración nativa con **Groq** para respuestas en milisegundos.
- **Desglose Neuronal:** Capacidad de fragmentar proyectos complejos en subtareas lógicas.
- **Neural Link UI:** Interfaz moderna y minimalista desarrollada en React + Vite.
- **Arquitectura Robusta:** Backend escalable con Flask y persistencia de datos con SQLAlchemy.
- **Contenedorización Total:** Despliegue simplificado mediante Docker y Docker Compose.

## 🛠️ Stack Tecnológico
- **Frontend:** React, TypeScript, Tailwind CSS, Vite.
- **Backend:** Python, Flask, SQLAlchemy.
- **IA:** Motor Llama 3 vía Groq Cloud API.
- **DevOps:** Docker, PowerShell Automation Scripts.

## 🚀 Guía de Uso
1. Configura tu API Key en el archivo .env.
2. Inicia el sistema: `powershell -File scripts/start.ps1`
3. Accede a: http://localhost:3000
